# Java POO

Exercícios e exemplos de Programação Orientada a Objetos em Java.

## Conteúdo
- Classes e objetos
- Herança
- Polimorfismo
- Encapsulamento

## Requisitos
- JDK 8 ou superior

## Como executar
Compile e execute os arquivos `.java` com seu compilador favorito ou usando uma IDE como Eclipse ou IntelliJ.
